# Handwritten-Digit-Recognition-using-Neural-Network
Description
This project implements a Handwritten Digit Recognition system using a Neural Network with TensorFlow/Keras. The model classifies handwritten digits (0–9) from 28×28 grayscale images using the MNIST dataset.

Dataset
MNIST handwritten digits dataset
60,000 training images
10,000 test images

Model Architecture
Flatten layer
Dense (128 neurons, ReLU)
Dense (64 neurons, ReLU)
Dense (10 neurons, Softmax)

Training
Optimizer: Adam
Loss: Categorical Crossentropy
Epochs: 10
Batch size: 32

Results
The model achieves approximately 97% accuracy on digit classification.

Technologies
Python, TensorFlow/Keras, NumPy, Pandas, Matplotlib, Scikit-learn
